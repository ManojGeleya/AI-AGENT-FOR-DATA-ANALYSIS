# Autonomous AI Agent for Data Analysis - Frontend Prototype

This is the frontend implementation of the Autonomous AI Agent for Data Analysis platform.

## Features
- Modern Corporate UI Design
- File Upload Interface
- Simulated AI Agent Processing Status
- Interactive Data Dashboard
- Visualization placeholders
- **Internal AI Agent Integration (No User API Key required)**
- **User Authentication (Email/Password)**

## Tech Stack
- React 18
- Tailwind CSS
- Chart.js
- Lucide Icons
- Internal `invokeAIAgent` for Logic

## Updates
- Removed dependency on external Gemini API keys from users.
- Integrated internal AI agent for analysis and chat.
